Лабораторна робота №2

HTML

HTML (HyperText Markup Language) - це стандартна мова розмітки для створення веб-сторінок. Використовуючи HTML, ми можемо описувати структуру та вміст нашої сторінки, включаючи заголовки, параграфи, зображення, посилання та багато іншого. HTML визначає, як інформація на вашому веб-сайті буде відображатися користувачам у браузері.

CSS

CSS (Cascading Style Sheets) - це мова стилізації, яка використовується для керування зовнішнім виглядом веб-сторінок, створених за допомогою HTML. CSS дозволяє нам встановлювати кольори, шрифти, розміри, відступи та інші стилізаційні параметри, щоб зробити наш вміст більш привабливим та організованим.

JavaScript

JavaScript - це мова програмування, яка використовується для надання веб-сторінкам інтерактивності. Ми можемо використовувати JavaScript для створення відгуків на події, які виникають на сторінці, такі як натискання кнопок, введення користувачем даних та багато іншого. JavaScript допомагає робити наші веб-сторінки більш динамічними та корисними для користувачів.
